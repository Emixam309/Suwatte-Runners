# [Kavita](https://www.kavitareader.com/) Runner for [Suwatte](https://suwatte.mantton.com/)

Connect your Kavita server to Suwatte and read your comics and manga seamlessly.

## Features

- Browse your Kavita libraries
- Read chapters directly in Suwatte
- Two-way progress sync between Kavita and Suwatte
- Partial support for Kavita's reading profiles
